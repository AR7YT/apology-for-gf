🎵 Background Music Instructions 🎵

To play "Those Eyes" by New West as the background music:

1. Download or locate your MP3 file for the song "Those Eyes" (by New West).
2. Copy the MP3 file.
3. Paste the file directly into this folder (c:\Users\asus\Documents\apology\music\).
4. Rename the file exactly to:
   those_eyes.mp3

Once you place the file here with the name "those_eyes.mp3", opening the website will automatically load and play it when you interact with the page!
